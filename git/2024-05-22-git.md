title: Git 사용법

* git bash에서 
cd 원격 경로
git clone 클론할 주소

* vscode 에서
원격 경로에 .git 생성됨, 그걸 선택하면 vscode 열림


## 키 발급 
* vscode 터미널에서
```
git config --global user.name "put your name"
git config --global user.email "put your email"
```

## 한글이 깨지는 경우
```
git config --global core.quotepath off
```
