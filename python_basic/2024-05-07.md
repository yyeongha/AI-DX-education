# pyqt 이용한 crawling

## <구현순서>
1. 화면설계 (pyqt)
2. 멀티윈도우(메인, 크롤링, 데이터 분석)
3. crawling
4. wordcloud 구현


## <맨 처음 확인할 사항>
* 버전 호환 되는지 확인한다. 
-> pypi ~ 로 확인


## <프레임워크 확인>
pypi wordcloud


## 가상환경 만들기 (p310_crawling)

## 화면 설계
* 메인            
main.py        
ui/ui_main.ui  
-> 메인은 메인윈도우 나머지는 다이얼로그로 만듦

* 크롤링
crawling.py
ui/ui_crawling.ui

* wordcloud
wordcloud.py
ui/ui_wordcloud.ui


## Main window 생성
* 이름 : ui_main.ui
* 네이버 지식인 크롤링 : btn_search
* 워드클라우드 데이터 분석 : btn_wordcloud
* 종료 : btn_close

![mainui](https://github.com/yyeongha/AI-DX-education/blob/main/python_basic/2024-05-07-img/mainui.png?raw=true)
